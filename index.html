What to paste into the GPT

The button Copy prompt for GPT creates a single prompt that tells the GPT:

  • Use only the provided reference text.
  • Generate the exact MCQ count requested.
  • Use A–D only, one best answer, no all/none-of-the-above.
  • Infer level automatically and match wording/difficulty.
  • Provide rationales for correct and incorrect options.
  • Output as JSON so this page can render the quiz.

If your GPT already has these rules embedded, students just paste the reference text + MCQ count and then paste the JSON back here.
